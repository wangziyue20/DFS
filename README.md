# Distributed File System

一个基于 C++ 开发的轻量级分布式文件系统，支持 **大文件分片上传与下载、数据冗余副本、节点管理与心跳检测、高并发性能测试与优化**。
本项目旨在模拟生产环境下的分布式存储服务架构，适用于学习、研究和简化原型的快速构建。

##  项目亮点

-  **Master + DataNode 架构**：自动调度数据分片与副本，支持节点心跳上报与失效检测  
-  **大文件上传下载**：基于分片策略处理上传与合并下载，避免单点瓶颈  
-  **多副本容错机制**：每个分片保存多份副本，提高数据可靠性  
-  **上传下载服务分离**：UploadServer/DownloadServer 提供统一 API 接口，隐藏底层节点细节  
-  **性能评估支持**：支持 Python 多线程测试脚本、Locust 高并发压测、日志记录性能数据  
-  **Redis 缓存优化**：下载路径使用缓存加速，未命中则自动回源 MySQL 并回写缓存
